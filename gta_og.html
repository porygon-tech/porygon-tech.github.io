GTA

Welcome

Graph-To-Adjacency (GTA) is a tool that helps you to edit and quickly translate from graph topologies to adjacency matrices and vice versa.
Although pretty straightforward to use, below you will find some pro hints.

How does this work?

Use the buttons to create or destroy selected nodes and links. If you select an existing node and click "Add Node", the newly generated nodes will link to it. If you select more than one node (long-click them or click while pressing Ctrl), the new nodes will link to these. To create a link, first select two nodes. If you click "Remove Link" while selecting a node, all its links will be removed. The navigation buttons help you scroll and zoom in case you cannot use your mouse.
