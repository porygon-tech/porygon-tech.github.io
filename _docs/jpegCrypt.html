Spread-spectrum QIM robust image steganography

Steganography robust to JPEG compression (it even survives WhatsApp's aggressive loss). Robustness comes at the cost of reduced vessel capacity, so you'll be only able to embed short sentences. Make sure you use the same seed for embedding and recovering!

Relevant infosec notes for nerds

This is keyed steganography, which doesnt make it cryptographically secure. The payload is still in plaintext before embedding, so an attacker who guesses the seed can fully recover message. It's on you to encrypyt the payload before embedding.

This site is static, so it runs client-side:

  • Your image never uploads
  • The message is never sent anywhere
  • The encoded result is generated locally in your computer
  • No server sees the data
However, be aware that the algorithm, the constants, and default seed are public. This implies some flaws for the purists.
